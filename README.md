A directory of repositories that I contributed code to.

A star :star: marks repositories where I am the main or sole author.

I am a contributor to repositories with no star. A few with no star are maintained forks, ports, etc.

## Quantum Computing/Information

### Qiskit

* **[qiskit](https://github.com/Qiskit/qiskit)** - Qiskit is an open-source SDK for working with quantum computers at the level of extended quantum circuits, operators, and primitives.
* **[qiskit-ibm-runtime-c](https://github.com/Qiskit/qiskit-ibm-runtime-c)** - IBM Quantum Platform API Client for Qiskit's C API
* **[documentation](https://github.com/Qiskit/documentation)** - The documentation content home for https://quantum.cloud.ibm.com/docs
* **[qiskit-tutorials](https://github.com/Qiskit/qiskit-tutorials)** - A collection of Jupyter notebooks showing how to use the Qiskit SDK
* **[samplomatic](https://github.com/Qiskit/samplomatic)** - Sample randomizations of Qiskit quantum circuits (e.g. Pauli twirling and noise injection) using declarative, composable annotations.
* **[qiskit-device-benchmarking](https://github.com/qiskit-community/qiskit-device-benchmarking)**
* **[qiskit-experiments](https://github.com/qiskit-community/qiskit-experiments)** - Qiskit Experiments
* **[qiskit-nature](https://github.com/qiskit-community/qiskit-nature)** - Qiskit Nature is an open-source, quantum computing, framework for solving quantum mechanical natural science problems.
* **[qiskit-aqua](https://github.com/qiskit-community/qiskit-aqua)** - Quantum Algorithms and Applications
* :star: **[QiskitRuntime.jl](https://github.com/jlapeyre/QiskitRuntime.jl)** - Julia API to IBM Qiskit Runtime
* **[qiskit-metapackage](https://github.com/Qiskit/qiskit-metapackage)** - Qiskit is an open-source SDK for working with quantum computers at the level of circuits, algorithms, and application modules.

### OpenQASM
* :star: **[openqasm3_parser](https://github.com/jlapeyre/openqasm3_parser)** - Parser and semantic analyzer for the OpenQASM3 language
* **[openqasm](https://github.com/openqasm/openqasm)** - Quantum assembly language for extended quantum circuits
* **[qiskit-qasm3-import](https://github.com/jlapeyre/qiskit-qasm3-import)** - Importer from OpenQASM 3 to Qiskit's QuantumCircuit
* **[Quasar.jl](https://github.com/kshyatt-aws/Quasar.jl)** - Qu(antum)as(sembly)(Lex/Pars)ar

* **[qe-compiler](https://github.com/openqasm/qe-compiler)** - An MLIR-based compiler dynamic circuit compiler for real-time control systems supporting OpenQASM 3
* :star: **[openqasm-rust](https://github.com/jlapeyre/openqasm-rust)** - QASM 3.0 parser for the Rust language.


### Gate Synthesis
* **[rsgridsynth](https://github.com/qiskit-community/rsgridsynth)** - high-precision Rust-based synthesizer that decomposes single-qubit Z-axis rotations into Clifford+T gate sequences, using number-theoretic algorithms and geometry-of-numbers methods.
* **[pygridsynth](https://github.com/quantum-programming/pygridsynth)** - Python version gridsynth program computes approximations of Z-rotations over the Clifford+T gate set

### Quil

* **[qvm](https://github.com/quil-lang/qvm)** - The high-performance and featureful Quil simulator.
* **[pyquil](https://github.com/rigetti/pyquil)** - A Python library for quantum programming using Quil.
* **[forest-benchmarking](https://github.com/rigetti/forest-benchmarking)** - A library for quantum characterization, verification, validation (QCVV), and benchmarking using pyQuil.
